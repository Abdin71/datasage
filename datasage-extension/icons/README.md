# DataSage Chrome Extension Icons

The extension requires icons in the following sizes:
- 16x16 (icon-16.png)
- 32x32 (icon-32.png)
- 48x48 (icon-48.png)
- 128x128 (icon-128.png)

You can:
1. Use the existing icons from the `/out` folder
2. Create custom icons using a tool like GIMP, Photoshop, or Figma
3. Use an online icon generator

For now, you can rename and copy the existing icons from the out folder.
